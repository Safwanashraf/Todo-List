# Todo Application  

A simple todo application built with React. This app allows users to create, update, and delete tasks effortlessly.

## Technologies Used  

- **React**  
- **React Router**
- **CSS Modules / Styled Components**
- **UUID**

## Features  

- Create new todos  
- Mark todos as complete  
- Edit existing todos  
- Delete todos
